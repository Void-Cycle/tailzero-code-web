Features / ฟีเจอร์

เบา เร็ว และใช้งานง่าย — ไม่มี bloat ไม่มีของที่ไม่จำเป็น

📁
File Explorer
VS CODE-STYLE FILE TREE

เปิดโฟลเดอร์และดู file tree แบบ VS Code กางโฟลเดอร์ได้ แยก folder/ไฟล์ชัดเจน

📄
Multi-Tab Editor
OPEN MULTIPLE FILES AT ONCE

เปิดหลายไฟล์พร้อมกันด้วย tab system แสดง line number และสถานะ unsaved

💾
แก้ไขและบันทึก
EDIT & SAVE WITH CTRL+S

แก้ไขโค้ดได้โดยตรง บันทึกด้วย Ctrl+S รองรับทุกภาษา undo/redo ครบ

⚡
เบาและเร็ว
LIGHTWEIGHT — ONLY 1.94 MB

สร้างด้วย Tauri + Rust ขนาดแค่ 1.94 MB เปิดเร็ว ใช้ RAM น้อยกว่า Electron 10 เท่า
